Building BlueEye: A Modern Artist Discovery & Booking Platform

May was really fun! Just shipped (and still maintaining + advancing with new features) a full-stack platform connecting Indian performing artists with clients - from discovery to booking inquiries, all handled seamlessly.

This was my first proper freelance+ Next.js project and honestly, it turned out really good. I handled everything - from designing and frontend states to backend architecture and deployment/hosting. That full-stack ownership was honestly the best learning experience.


Core Tech Stack:

- Next.js 14 + TypeScript for type-safe, server-rendered performance
- MongoDB + Mongoose for flexible, scalable data architecture
- NextAuth.js with OAuth support for secure multi-role authentication
- TailwindCSS + shadcn/ui for responsive, accessible design
- Framer Motion & GSAP for premium animations & interactions
- ImageKit CDN for optimized media delivery at scale
- Redis for high-performance caching & session management
- Resend for transactional email & notifications


Smart Features Built In:

- Live artist search suggestions with full-text MongoDB indexing
- Advanced multi-filter search (category, city, genre, language)
- Artist profiles with rich media galleries & detailed bios
- Intelligent switch account page (multi-role user support)
- Booking inquiry system with admin status management
- Event registration with real-time user tracking
- JSON bulk import for rapid catalog onboarding
- Real-time platform statistics dashboard
- Comprehensive SEO with structured data & sitemap generation
- Admin dashboard with live edit capabilities


What I'm Most Proud Of:

The whole architecture, honestly. MongoDB text indexing combined with intelligent filtering creates a smooth discovery experience. Plus, leveraging environment-based configurations for a truly multi-tenant ready platform. It's clean, scalable, and actually maintainable.


Current Phase: 
SEO optimization, seamless communication flows, and a few more features for better UX


In the Pipeline:
- Auto-backup system for booking inquiries (dual table, backup-on-demand with UI feedback)
- Personal inquiry tracking for artists (view-only, admin manageable)
- Security layer: double confirmation popups for deletions + OTP-gated bulk deletes via Resend
- More artist profiles and data enrichment
- Event management expansion
